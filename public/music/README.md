# Music Files

## How to Add Your Song

1. **Convert your song to MP3 format** (if it isn't already)
2. **Rename the file** to: `tu-itni-khoobsurat-hai.mp3`
3. **Place the file** in this folder: `public/music/`

## File Requirements

- **Format**: MP3 (recommended for web compatibility)
- **Size**: Keep under 10MB for faster loading
- **Quality**: 128kbps or 192kbps is good for web

## Current Setup

The website is configured to play:
- **Main song**: `tu-itni-khoobsurat-hai.mp3`
- **Auto-play**: Starts after user's first interaction (click/tap)
- **Loop**: Song repeats continuously
- **Volume**: Fixed at 40% (optimal background level)
- **Control**: Simple play/pause button only
